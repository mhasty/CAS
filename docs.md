# Product Notes

## Positioning
Caarps is a balanced appraisal workflow platform for appraisers, AMCs, and lenders. It should not be framed as anti-lender or anti-AMC. It is pro-speed, pro-transparency, pro-compliance, and appraiser-friendly.

## Borrower visibility
Borrowers are not the client. Borrower interaction should be limited to controlled scheduling or document coordination if enabled by the lender/client. Borrowers should not see appraisal reports, lender notes, review findings, accounting, or order workflow details.

## Pricing strategy
- No monthly appraiser fee by default.
- Transparent low tech fees, capped around $9.99 per applicable transaction.
- Monetize enterprise features through lender/AMC workflow automation, analytics, integrations, white-label portals, API access, compliance tooling, and review automation.

## Production caution
Before regulated use, consult appraisal compliance, lender policy, GLBA/privacy, data retention, state requirements, AI disclosure obligations, and information-security counsel.
